State Robot

Per the rules of the University of California, Santa Cruz, the source code for this project can not be shared publicy. To view the source code please send me an email, which can be found in the contact section of the website, and state a reason for requesting source code.

Overview

In this assignment we were tasked with writing a basic client-server application in C++. The application uses serialized messages to communicate across a TCP socket connection.

Program Description

We were provided with a skeleton to fill in for both the client and server sides of the program. The server will recieve messages, desearilize them, and output the message on the server. The client will send protobuf messages to the server.
